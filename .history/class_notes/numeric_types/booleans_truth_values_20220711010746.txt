   "source": [
    "Objects have Truth Values\n",
    "\n",
    "All objects in Python have an associated truth value.\n",
    "\n",
    "We already saw this with integers.      (although to be fair, bool is a subclass of int)\n",
    "\n",
    "But, this works the same for any object.\n",
    "\n",
    "In general, the rules are straightforward\n",
    "\n",
    "\n",
    "Every object has a True truth value, except:\n",
    "\n",
    ".   None\n",
    ".   False\n",
    ".   0 in any numeric type (e.g. 0, 0.0, 0+0j, ...)\n",
    ",   empty sequences (e.g. list, tuple, string, ...)\n",
    ".   empty mapping types (e.g. dictionary, set, ...)\n",
    ".   custom classes that implement a __bool__ or __len__ method that returns False or 0\n",
    "\n",
    "which have a False truth value.\n",
    "\n",
    "\n"
   ]
  },
  {
   "cell_type": "markdown",
   "id": "554f2c5f",
   "metadata": {},
   "source": [
    "        Under the hood\n",
    "\n",
    "Classes define their truth values by defining a special instance method method:\n",
    "\n",
    "        __bool__(self)      (or __len__ )\n",
    "\n",
    "Then, when we call bool(x)  Python will actually executes x.__bool__()\n",
    "                                                    or __len__  if__bool__ is not defined\n",
    "                                                    if neither is defined, then True\n",
    "\n",
    "Example: Integers\n",
    "\n",
    "def __bool__(self):\n",
    "    return self != 0\n",
    "\n",
    "When we call bool(100) Python actually executes 100.__bool__()\n",
    "and therefore returns the result of 100 != 0 which is True\n",
    "When we call bool(0)   Python actually executes 0.__bool__()\n",
    "and therefore returns the result of 0 != 0      which is False\n",
    "\n",
    "\n",
    "Examples\n",
    "\n",
    "bool([1, 2, 3]) -> True\n",
    "                          _______\n",
    "bool([]) -> False        |    if my_list:\n",
    "                _________|       # code block\n",
    "bool(None) -> False      |\n",
    "                         |    code block will execute if and only if my_list is\n",
    "bool('abc)  -> True      |    both not None and not empty\n",
    "bool(') -> False         |\n",
    "                         |    this is equivalent to:\n",
    "                         |    if my_list is not None and len(my_list) > 0:\n",
    " bool(0) -> False        |__     # code block\n",
    "bool(0 + 0j) -> False\n",
    "bool(Decimal('0.0)) -> False\n",
    "\n",
    "bool(-1) -> True\n",
    "bool(1 + 2) -> True\n",
    "bool(Decimal('0.1)) -> True\n",
    "\n"
   ]
